Frequently asked questions

What’s available offline?

Campground data itself is stored on the device, so it is always available offline. Obviously any internet resource such as websites would not be available offline. The app uses the device’s native map technology so map availability depends on the device. For example for Android, Google Maps allows you to save maps of an area for offline use which would be available in Kampnik.

What do the marker colors mean?

The marker colors are categories of campgrounds. In the home screen tap the button and open the legend to see a full list of campground categories.

How is distance measured?

Distance is measured by airplane distance, so a direct line from one point to another. It is not turn-by-turn or driving distance.
